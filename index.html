Autonomous grading system for Sinhala Language Essays of grade 5 students

As technology advances, education is transforming, especially in exam evaluation. Traditional grading is often time-consuming and subjective, but automated systems using Artificial Intelligence (AI) and Machine Learning (ML) are changing this. This research focuses on developing an autonomous grading system for Sinhala language tests using Natural Language Processing (NLP). Given the unique characteristics of Sinhala, existing systems designed for English may not effectively assess Sinhala essays. This research aims to create a tool that understands Sinhala’s morphology and syntax, improving grading accuracy and efficiency. By streamlining the assessment process, this system will reduce instructors' workload and ensure fair evaluations, ultimately enhancing the learning experience for students.

Literature Survey

The goal is to enhance grading accuracy and improve the educational experience for students in Sri Lanka.

Proposed Problem

This study aims to develop an NLP-based autonomous grading system specifically designed to evaluate Sinhala essays written by grade 5 students. The goal is to create a tool that not only reduces the time and effort required by teachers but also offers accurate and unbiased assessments, compensating for the linguistic and structural nuances of the Sinhala language.

Proposed Solution

The proposed solution involves designing an NLP model that processes Sinhala text, correcting spelling, simplifying complex words, and evaluating grammatical accuracy. By leveraging techniques like tokenization, part-of-speech tagging, and syntactic analysis, the system will assess key aspects such as coherence, originality, and language use in student essays. The system will also offer feedback to enhance learning

Main Objective

The primary goal of this project is to create an autonomous grading system for Sinhala language essays written by grade 5 students. This system aims to evaluate essays in an unbiased, efficient, and consistent manner using natural language processing (NLP) techniques specifically designed for the Sinhala language. The grading system will streamline the assessment process for educators, reducing the time and effort required for manual grading. It will also improve accuracy by addressing common issues like spelling mistakes, grammatical errors, and overly complex language, ultimately providing more insightful feedback to students to aid their learning and development in Sinhala writing.

Methodology

The autonomous Sinhala essay grading system operates through a mobile interface where users submit essays for evaluation via a REST API, which connects to a backend database storing essential resources. After user input is preprocessed for NLP analysis through various linguistic tasks, identification modules detect native words, correct misspellings, identify grammar mistakes, and recognize key terms. Algorithms then analyze these aspects, providing corrections and feedback. Finally, predicted results, including evaluations and suggestions, are sent back to the user, with the processed data stored for continuous improvement of the system.

Research Gap

While automated grading systems for English and other major languages are well-established, the absence of robust systems tailored to the Sinhala language is a significant gap. Sinhala's unique syntactic structures, rich morphology, and orthographic variations make it a challenging language for NLP. Existing solutions fail to adequately address these complexities, particularly in an educational context involving young learners.
